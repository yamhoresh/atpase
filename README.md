# ATPase apps website

Single-page marketing site for ATPase apps, an enterprise-ready engineering partner delivering custom mobile and web products. Built with plain HTML/CSS so it can be deployed instantly on GitHub Pages.

## Local preview

```bash
python3 -m http.server 8000
```

Visit http://localhost:8000 to view the site.

## Deploying to GitHub Pages

1. Commit the latest changes on the `main` branch.
2. Push to the GitHub repository.
3. In the repository settings, enable GitHub Pages for the `main` branch (root folder). The site will be live at `https://<username>.github.io/<repo>/` within a minute.
